# 图片转base64

> 将本地图片或者网络图片转换为base64，并调用到系统的复制粘贴板。

```shell
$ asd imgtl [options] <imgStr> 
```

#### options参数, __该命令参数必选__

| name |       alias     |          desc       |
| :--: | :-------------: | :---------------------: |
|  -ib  |    --imgtobase64    |      图片路径转base64  |
|  -ub  |    --urltobase64    |   图片链接转base64 |

#### imgStr
> 本地图片路径,或者网络图片链接.
